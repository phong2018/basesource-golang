## Quick daily command (run this first every morning)

```bash
# Cluster health at a glance
kubectl get pods -n basesource
kubectl get hpa -n basesource
kubectl get pvc -n basesource
kubectl top pods -n basesource    # requires metrics-server
```

Expected: all pods `1/1 Running`, PVCs `Bound`, HPA replicas at min.

## Architecture reminder

```
User → Ingress → basesource-api (×2) → MySQL
                                      → outbox_events

Worker (×2) → KafkaOutboxRelay    → Kafka  → KafkaConsumer
            → RabbitMQOutboxRelay → RabbitMQ → NotifConsumer

Node agents (1 per node):
  fluent-bit    → collects all container logs
  node-exporter → exposes CPU/memory/disk metrics at :9100/metrics
```
